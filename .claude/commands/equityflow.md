# /equityflow — A股金融市场资金流向追踪

触发 `equity-capital-flow` skill，八维度追踪金融市场增量资金，输出月度备忘录 Section 2。

## 用法

```
/equityflow [月份]
```

示例：
```
/equityflow 2026年5月
/equityflow 5月
```

## 运行内容

- 2.1 开户数（情绪先行）
- 2.2 偏股基金发行（增量主力 + 近12月序列）
- 2.3 基金市场规模结构（各类份额时序 + 环比变动表）
- 2.4 宽基ETF净申购（国家队 vs 市场自发）
- 2.5 行业主题ETF净申购（轮动方向）
- 2.6 两融（融资净买入 + 历史分位）
- 2.7 股权融资（流出端）
- 2.8 外资（用户提供 / 标「数据未提供」）
- 2.9 月度净流量粗估汇总表

## 说明

单独运行时只输出 Section 2；完整备忘录请用 `/capitalflow`。
