# Skill: monthly-capital-flow-memo

> 对应教程 Ch48。本 Skill 是整合层，不独立做分析，只拼合 Ch46 / Ch47 的输出并补写总结。

## 输入

- `period`：数据期间（如 2026年5月）
- `data_cutoff`：数据截止日期（如 2026年5月末）
- `write_date`：备忘录撰写日期
- Ch46（macro-credit-flow）的输出（Section 1 全文）
- Ch47（equity-capital-flow）的输出（Section 2 全文）

## 输出

```
output/日常记录/XXXX年X月社会资金流向备忘录.md
```

## 步骤

### 步骤 1：运行 macro-credit-flow（Ch46）

触发 macro-credit-flow skill，获得 Section 1（信用货币环境）全文。

### 步骤 2：运行 equity-capital-flow（Ch47）

触发 equity-capital-flow skill，获得 Section 2（金融市场资金结构）全文。

### 步骤 3：写封头 + 总结

封头格式：

```markdown
# XXXX年X月 社会资金流向备忘录

**日期**：YYYY-MM-DD  
**数据截止**：XXXX年X月末

> 本备忘录为同步复盘类文件。资金流向是同步指标，核心用途是：① 对当前市场结构有全面了解；② 理解每类投资者的目标和约束。不做单点预测，仅供研究参考，不构成投资建议。
```

### 步骤 5：输出文件

写入 `output/日常记录/XXXX年X月社会资金流向备忘录.md`，文件名格式固定。

## 检查清单

- [ ] 封头日期和数据截止填写正确
- [ ] 总结的一句话结论不超过30字，无"必然/一定"
- [ ] 宏观层表格数据与 Section 1 一致
- [ ] 市场层净额计算有 ⚠️ 提示（含重叠口径）
- [ ] 行为层至少写出2个结构性变化，不能只是数字堆砌
- [ ] 底线判断为现状描述，不做单点预测
- [ ] 数据缺口说明完整，不静默跳过
- [ ] 输出路径正确（output/日常记录/）
