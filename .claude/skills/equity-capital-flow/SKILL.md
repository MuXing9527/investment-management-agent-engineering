# Skill: equity-capital-flow

> 对应教程 Ch47。分析框架参见 Ch47 正文；本文件为操作步骤速查。

## 输入

- `period`：数据期间（如 2026年5月）
- 主要数据文件（`data/` 文件夹）：
  - 开户数：`投资者开户统计（上交所）-统计.xlsx`
  - 基金发行：`基金发行统计.xlsx`
  - 基金市场规模：`基金市场规模(同花顺统计).xlsx`
  - ETF 净申购：`ETF净申赎资金统计.xlsx` + `ETF规模统计(按指数).xlsx`
  - 两融：`市场交易统计.xlsx`（融资余额 + 融资净买入）
  - 股权融资：`股权融资规模(同花顺统计).xlsx`
- 外资数据：用户手动提供（EPFR 口径），不从其他来源抓取

## 输出

月度备忘录 Section 2，含八个子节：

```
Section 2  金融市场资金结构
├── 2.1 开户数（情绪先行）
├── 2.2 偏股基金发行（增量主力 + 近12月序列）
├── 2.3 基金市场规模结构（各类份额时序 + 当月环比变动表）
├── 2.4 宽基ETF净申购（国家队 vs 市场自发）
├── 2.5 行业主题ETF净申购（轮动方向）
├── 2.6 两融（融资净买入 + 余额 + 历史分位）
├── 2.7 股权融资（IPO + 定增，流出端）
├── 2.8 外资（北向 / 主动 EPFR / 被动，有则填，无则标「数据未提供」）
└── 2.9 月度净流量粗估汇总表
```

## 步骤

### 步骤 1：开户数

- [ ] 读取上交所个人/机构新增开户数，与上月及3年历史均值对比
- [ ] 定性判断：情绪处于峰值 / 正常 / 低谷区间

### 步骤 2：偏股基金发行

- [ ] 读取当月偏股基金（股票型+混合型）合计发行份额
- [ ] 附近12个月序列图，判断趋势（上升/平稳/下降）
- [ ] 估算当月直接入市规模（建仓比例约50-60%首月）

### 步骤 3：基金市场规模结构

- [ ] 读取全市场基金总份额（各类型：货基/债基/股票型/混合型/QDII/FOF/REITs）
- [ ] 生成各类基金份额时序图（近13个月）+ 占比时序图
- [ ] 制作当月 vs 上月环比变动表（份额变化 + 净值变化，分主要子类细拆）
- [ ] **结构解读**：重点说明被动指数型股票基金、偏股混合、货基、FOF/REITs 的变化方向
- [ ] 判断："ETF赎回的钱去哪了"—— 是否有跨类型轮动迹象

### 步骤 4：宽基ETF净申购

- [ ] 读取宽基ETF（规模类ETF）月度净申购数据
- [ ] **国家队识别**：判断是否有异常放量（> 200亿/日）、主要集中在沪深300/上证50、与下跌时间点吻合
- [ ] 输出：含国家队估算的净申购 vs 剔除国家队后的市场自发资金
- [ ] 附自2023年初累计净流入走势图

### 步骤 5：行业主题ETF净申购

- [ ] 读取行业ETF + 主题ETF + 跨境ETF 净申购
- [ ] 判断轮动方向（哪些行业/主题仍有资金流入）

### 步骤 6：两融

- [ ] 读取融资余额（月末）+ 融资净买入（4周累计）
- [ ] 与上月对比，附历史序列，判断分位水平
- [ ] 注意：用「融资净买入」而非「融资余额净变化」（余额含价格成分）

### 步骤 7：股权融资

- [ ] 读取当月 IPO 募资（金额+单数）+ 定增募资（金额+单数）
- [ ] 合计流出规模，与近12个月比较，判断一级市场抽血压力

### 步骤 8：外资

- [ ] 若用户提供 EPFR 数据：填写主动/被动分项，说明流入/流出
- [ ] 若无数据：标注「数据未提供，EPFR 为基金口径」，不从其他来源抓取
- [ ] 解读结构：被动流出 vs 主动流出的比例，含义不同

### 步骤 9：净流量汇总表

按模板填写月度净流量粗估表，加注 ⚠️ 提示：此表为方向性参考，含重叠口径，不做精确加总。

## 目标与约束分析框架（核心视角）

每类资金不只报量，还要回答"他们为什么买、什么条件下会停"：

| 资金类型 | 目标 | 核心约束 | 持续条件 |
|---|---|---|---|
| 主动权益基金 | 跑赢基准 | 赎回压力 / 排名竞争 | 基准上涨 / 净申购 |
| 宽基ETF（市场自发）| 指数复制 | 赎回压力 | 市场预期改善 |
| 宽基ETF（国家队）| 稳市/防跌 | 政策指令 | 政策需要时，不可预设持续 |
| 两融 | 放大收益 | 维保比例 / 爆仓线 | 市场不大幅下跌 |
| 外资（主动）| 全球配置比例 | 汇率/地缘/风险偏好 | 人民币稳/溢价下降 |
| 外资（被动）| 跟踪指数权重 | MSCI/富时调仓窗口 | 指数权重维持 |

## 典型 Anti-pattern

| 错误做法 | 应该这样 |
|---|---|
| 只看当月发行数字，不看趋势 | 必须附近12月序列，判断趋势 |
| 不做基金规模结构拆解 | 基金市场规模结构是理解ETF赎回去向的关键视角 |
| 宽基ETF大额净申购不判断国家队 | 必须识别是否含国家队并做标注 |
| 理财/保险直接写「缺失」| 先网络检索，找不到才标「无」 |
| 外资不拆主动/被动 | 两者含义完全不同，必须区分 |
| 只报绝对量，不报历史分位 | 历史分位才能判断当前热度 |
| ETF赎回不分析资金去向 | 需做"钱去哪了"的交叉验证（回权益/固收/真实撤退） |
