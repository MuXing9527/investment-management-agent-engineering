# Skill: macro-credit-flow

> 对应教程 Ch46。分析框架参见 Ch46 正文；本文件为操作步骤速查。

## 输入

- `period`：数据期间（如 2026年5月，数据截止月末）
- 金融数据文件：`D:\My_Project\Strategys\vn-py-dev\examples\signal_analysis\results\macro_excel\金融.xlsx`
- 外贸数据：同目录 `外贸.xlsx`
- 央行操作：`data/bond_release/货币投放(回笼)统计.xlsx`
- 外汇占款：`data/bond_release/金融机构人民币信贷收支(月)_*.xlsx`（取最新日期）
- 债券发行：`data/bond_release/`（国债/地方债/政金债/同业存单各品种时间序列）
- 官方点评（可选）：`arcticle/` 文件夹，有则读，无则跳过

## 输出

月度备忘录 Section 1，含三个子层：

```
Section 1  信用货币环境
├── 1.1 资产端（资金需求侧）
│   ├── 社融总量与结构
│   ├── 信贷结构拆分
│   ├── 票据充量判断（必答）
│   └── 直接融资
├── 1.2 负债端（资金供给侧）
│   ├── M2 / M1 同比 + 近12月序列
│   ├── M1-M2 剪刀差
│   └── 存款结构（非银存款单独列出）
└── 1.3 机构行为层
    ├── 央行操作汇总表
    ├── 同业存单发行
    ├── 债券净供给分品种表
    ├── 银行资产投向（有图描述/无图标「待补充」）
    └── 官方点评 + 印证判断（若有）
```

## 步骤

### 步骤 1：资产端（资金需求侧）

- [ ] 读取社融当月新增总量 → 同比多增/少增判断
- [ ] 拆分社融结构：政府债 / 贷款 / 企业债 / 非标
- [ ] 拆分信贷结构：企业短贷 / 企业中长贷 / 居民短贷 / 居民中长贷 / 票据融资
- [ ] **票据充量判断**：票据融资规模 vs 企业中长贷背离程度，判断是否充量并明确结论
- [ ] 生成近24个月分项图

### 步骤 2：负债端（资金供给侧）

- [ ] 读取 M2 / M1 同比，附近12月序列，判断趋势
- [ ] 计算 M1-M2 剪刀差，说明方向与历史位置
- [ ] 读取存款结构：居民 / 企业 / 政府 / **非银存款**（非银单独列出，是 equity-capital-flow 的上游接口）

### 步骤 3：机构行为层

- [ ] 读取央行操作：OMO + 买断式逆回购 + MLF + POMA，填写汇总表（含到期/投放/净额）
- [ ] 解读净投放/净回笼：区分主动收紧 vs 被动到期，说明期限结构含义
- [ ] 读取同业存单：当月发行量 + 近一年序列图
- [ ] 读取债券净供给：国债 / 地方债 / 政金债 / 信用债 / 同业存单
- [ ] 银行融出资金：有图则描述，无图标「待补充」，不能静默跳过
- [ ] 官方点评（若 arcticle/ 有文件）：摘要 + 正确部分 + 回避部分 + 逻辑漏洞分析

## 核心判断检查单

运行完成后，以下 8 个问题必须有明确答案：

| # | 问题 | 必须在哪里回答 |
|---|---|---|
| 1 | 社融是多增还是少增？主要驱动项？| 1.1 资产端小结 |
| 2 | 票据是否在冲量？如何判断？| 票据充量分析段 |
| 3 | M1-M2 剪刀差方向？附近12月趋势？| 1.2 负债端小结 |
| 4 | 非银存款增还是减？幅度多大？| 1.2，单独列出 |
| 5 | 央行本月净投放/净回笼？主动还是被动？| 1.3 机构行为层 |
| 6 | 债券净供给是否形成供给压力？| 1.3 债券净供给 |
| 7 | 银行融出/买债图是否处理？| 1.3（「待补充」不算跳过）|
| 8 | 官方点评与数据面是否一致？| 1.3 官方点评段 |

## 典型 Anti-pattern

| 错误做法 | 应该这样 |
|---|---|
| 只看社融总量，不拆结构 | 必须拆到企业/居民/政府/票据四层 |
| 票据冲量不下结论 | 背离明显时必须明确写"冲量/未冲量" |
| 非银存款不单独列 | 是下游权益分析的关键接口，不能合并 |
| OMO 只看总量，不看期限 | MLF 1Y vs 逆回购 7D 比例揭示期限偏好 |
| 只摘抄官方点评，不印证 | 必须做"官方定性 vs 实际数据"交叉判断 |
